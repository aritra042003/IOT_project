About Us

This project involves using a Raspberry Pi Pico W and a DHT11 sensor to detect temperature and humidity. Below you can find details about the hardware components used and their respective diagrams.

Project Description

This project is designed to monitor environmental conditions by using a Raspberry Pi Pico W microcontroller and a DHT11 sensor. The collected data is stored in a database and can be accessed via a web interface. The web application allows users to log in, view real-time data, and execute SQL queries for data analysis.

Hardware Components

  • Raspberry Pi Pico W
  • DHT11 Temperature and Humidity Sensor
  • Breadboard and Jumper Wires
  • Power Supply
